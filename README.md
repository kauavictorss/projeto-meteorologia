# 🌦️ Projeto Meteorologia - DevClub

![Versão](https://img.shields.io/badge/vers%C3%A3o-v1.1.0-blue)
![Status](https://img.shields.io/badge/status-conclu%C3%ADdo-brightgreen)

Uma aplicação de previsão do tempo moderna, rápida e responsiva, desenvolvida para consultar dados meteorológicos em tempo real de qualquer cidade do mundo.

---

## 🚀 Sobre o Projeto

Este projeto foi desenvolvido como parte de um desafio prático do **[DevClub](https://www.devclub.com.br)**, sob orientação do instrutor **Rodolfo Mori**. O objetivo foi aplicar conceitos fundamentais de desenvolvimento web, manipulação de DOM, consumo de APIs e design responsivo.

## ✨ Funcionalidades

- **Busca em Tempo Real:** Consulta a API da OpenWeatherMap para obter dados atualizados.
- **Interface Dinâmica:** O fundo da aplicação muda de acordo com a cidade pesquisada.
- **Responsividade:** Layout adaptado para uma experiência perfeita em computadores, tablets e smartphones.
- **Feedback Visual:** Spinner de carregamento integrado para melhorar a experiência do usuário durante a busca.
- **Atalhos:** Suporte para busca via clique no botão ou pressionando a tecla "Enter".

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação semântica da aplicação.
- **CSS3:** Estilização com conceitos de *Glassmorphism*, Flexbox e Media Queries.
- **JavaScript (ES6+):** Lógica de busca, consumo de API assíncrona (`fetch`, `async/await`) e manipulação dinâmica de elementos.
- **OpenWeatherMap API:** Fonte de dados meteorológicos.
- **Bootstrap Icons & CSS:** Ícones e componentes de feedback visual.
- **Unsplash.it:** Serviço de imagens dinâmicas para o fundo.

## 📂 Estrutura de Arquivos

```text
├── index.html   # Estrutura da interface
├── index.js     # Motor da aplicação (Lógica)
├── style.css    # Estilização e Responsividade
└── README.md    # Documentação do projeto
```

## ⚙️ Como Executar

1. Clone este repositório ou baixe os arquivos.
2. Certifique-se de ter uma chave de API válida da [OpenWeatherMap](https://openweathermap.org/api).
3. Insira sua chave no arquivo `index.js` na variável `chave`.
4. Abra o arquivo `index.html` em qualquer navegador moderno.
